HTML: Building Tables

Adding Table Headers

Table headers are an important part of structuring tables. As the name suggests, the <th> tag identifies a table cell as being a header and identifies row or column information. Like <td> elements, table headers can contain almost any content, although they cannot contain header, footer, sectioning content, or headings. A best practice is to keep headers short and descriptive, and avoid any uneccessary formatting. Several attributes such as the scope, abbr, and headers can help extend the meaning of headers and increase their accessibility.

Basic table elements:
Tag Name Description
<table> Table Identifies what follows as a table.
<tr> Table Row Establishes a new row within the table
<th> Table Header Identifies the contents of a cell as being a header for a group of cells
<td> Table Cell This element defines the individual table cells of an element
