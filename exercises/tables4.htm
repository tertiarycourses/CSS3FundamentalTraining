HTML: Building Tables

Adding Table Captions

Often, you may need to identify the contents of a table by giving it a title. While this can be accomplished by using a heading element prior to the table, a more sematic way would be to use a table caption. Captions are added to tables immediately after the opening table tag by using the <caption> element. Captions can be as complex as you need them to be, and can even contain formatted text and images. For the most part, however, captions should be brief and descriptive.

Although past versions of HTML contained attributes to position captions, they have been deprecated in HTML5 and all visual formatting of captions should be controlled through CSS.

Basic table elements:
Tag Name Description
<table> Table Identifies what follows as a table.
<tr> Table Row Establishes a new row within the table
<th> Table Header Identifies the contents of a cell as being a header for a group of cells
<td> Table Cell This element defines the individual table cells of an element
