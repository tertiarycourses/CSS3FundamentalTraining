HTML: Building Tables

Setting Table Attributes

Prior to HTML5, the <table> element had a number of attributes that primarily assisted with the presentation of the tablular data. These attributes are no longer part of the HTML specification, but will continue to be supported by almost all user agents for backwards compatibiity purposes. While not recommended as part of the authoring process, knowing how the border, cellpadding, cellspacing, and width attributes work is important for understanding legacy code, or for when styles are not available.

Tag Name Description
<table> Table Identifies what follows as a table.
<tr> Table Row Establishes a new row within the table
<th> Table Header Identifies the contents of a cell as being a header for a group of cells
<td> Table Cell This element defines the individual table cells of an element
