HTML: Building Tables

Adding content to tables

Unless restricted through attributes or styles, table cells expand to hold the content inside of them. This allows you to store multiple paragraphs of text, larger images, or objects like embedded video without fear of breaking the table. Having the content of one table cell be dramatically larger than other cells can negatively impact the readability of the table, so exercise care when constructing tables with large amounts of content, or objects with large dimensions.

Also, the <td> element contents can be styled accordingly, and all user agents have default styles for table content. Only format table content with elements like <p>, <h2>, and other elements when it is semantically appropriate to do so, and not for formatting purposes.

Tag Name Description
<table> Table Identifies what follows as a table.
<tr> Table Row Establishes a new row within the table
<th> Table Header Identifies the contents of a cell as being a header for a group of cells
<td> Table Cell This element defines the individual table cells of an element
